# 工作流2：收入模式（两层分配）

当用户报告收入时：

1. **记录收入**：`add_bill`，type 设为 income
2. **运行两层分配**：`income_allocation`，自动计算：
   - 第一层（刚性扣除）：固定账单 + 债务最低还款
   - 第二层（四包分配）：剩余可支配收入按比例分配
3. **展示分配结果**：将 HTML 写入 `output/allocation_{日期}.html`，用 `preview_url` 预览
4. **询问自定义**：用户可以调整比例，但【保命的钱】建议 ≥10%
5. **额外还款建议**：如果【悦己的钱】有富余，温柔建议："老大，如果这个月【悦己的钱】花不完，要不要拿一点去多还点债？早还完早自由~"
